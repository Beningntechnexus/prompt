# **App Name**: Prompt Explorer

## Core Features:

- Supabase Connection: Connects to Supabase with provided URL and key.
- Category Fetch: Fetches all categories from the 'categories' table in Supabase.
- Prompt Fetch: Fetches all prompts and associated category IDs from the 'prompts' table.
- Category Display: Displays categories in a grid layout.
- Prompt Display: Displays prompts associated with the selected category.
- Prompt Search: Filters prompts by keywords using a search bar.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a professional and clean look.
- Background color: Light gray (#F9FAFB), nearly white, to ensure excellent readability and to create a minimalist aesthetic.
- Accent color: A vibrant purple (#A855F7) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter' sans-serif for a modern, clean and neutral look.
- Simple, outline-style icons for categories and interactions.
- Grid layout for categories with a clean, responsive design.
- Smooth fade animations on hover and category selection.
